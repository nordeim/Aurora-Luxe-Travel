Aurora Luxe Travel - Luxury Travel Concierge Website
This is a production-grade luxury travel concierge website built with modern web technologies and exceptional attention to design detail.
Core Technology Stack
- Next.js 16.1.4 with App Router
- React 19.2.3 
- TypeScript 5.9.3 with strict mode
- Tailwind CSS v4 (CSS-first configuration)
- Framer Motion 12 for animations
- Radix UI primitives for accessibility
- Vitest for testing
Key Project Features
1. Distinctive "Celestial Noir" Design System
   - Deep space black backgrounds (#050506)
   - Aurora-inspired gradients (cyan → purple → magenta)
   - Champagne gold accents (#C9B896)
   - Custom glass morphism effects
   - Instrument Serif (headlines) + Geist (body) typography
2. Complete Component Library
   - 9 UI Primitives: Button, Input, Textarea, Select, GlowOrb, GlassPanel, SectionHeading, ChipGroup, Toast
   - 6 Section Components: Hero, Destinations, Experiences, Membership, Testimonials, ConciergeForm
   - 2 Layout Components: Navbar, Footer
   - Custom hooks: useScrollSpy, useReducedMotion
3. Data Layer
   - 6 curated luxury destinations
   - 5 signature experiences
   - 3 membership tiers
   - 3 client testimonials
4. Advanced Features
   - Smooth scroll navigation with active section tracking
   - Form validation with success/error handling
   - Accessibility compliance (reduced motion support, WCAG AAA)
   - Mobile-first responsive design
   - Performance optimized (Image optimization, modern CSS)

Architecture Highlights
- Modular component structure following separation of concerns
- TypeScript interfaces for all data models
- Tailwind v4 CSS-first approach (no tailwind.config.js)
- Custom design tokens defined in CSS @theme directive
- Proper accessibility with semantic HTML and ARIA labels
Business Context
Targets ultra-high-net-worth individuals (UHNWI) seeking bespoke travel experiences. The "Beyond First Class" positioning differentiates it from standard luxury travel offerings.
This is a showcase project demonstrating avant-garde UI design principles, technical excellence, and production-ready code quality. It's designed to be both visually striking and technically robust, avoiding generic "AI slop" aesthetics through intentional design decisions.
